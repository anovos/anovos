# Anovos Examples

To try the examples, run

```bash
./create_anovos_examples_docker_image.sh
sudo docker run -p 8888:8888 anovos-examples:latest
```

and open the link to `http://127.0.0.1:8888/?token...` generated by the Jupyter NotebookApp:
